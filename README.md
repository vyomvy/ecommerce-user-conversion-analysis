📊 Ecommerce Customer Behavior Analysis

By Sant (Vyom) Kumar Sahu

This project analyzes an ecommerce dataset to uncover insights about revenue trends, customer purchase behavior, and conversion patterns.

🔍 Key Analysis Performed
Revenue analysis (total + monthly trends)
Order status impact (completed vs cancelled)
Customer purchase behavior
Average Order Value (AOV)
Time from signup to first purchase
Distribution analysis (conversion timing)
📈 Key Insights
Majority of users (~63%) purchase before signup, indicating a checkout-first journey
Among post-signup users, conversion time is highly right-skewed
Revenue shows clear monthly variation with peak periods
AOV provides a baseline for upsell and cross-sell strategies
